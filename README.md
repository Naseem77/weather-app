 <h1 align="center">
  Weather App
</h1>

Weather app in Full Stack - client using OOP and MVC design, server access an external API, and save data to a MongoDB database with CRUD operations.

# Technologies:
- OOP
- MVC
- CURD
- MongoDB/Atlas
- jQuery
- Express
- urllib
- API's
- Handlerbars
